    Kiss on Java

    Kiss on JavaはJava実装です。ただしFKISS2までの対応です。

  • KissLoader on Java 1.1.1 (C) 1996,1998-9,2002,2004

    KOJ1.1.1は、JREが、Java1.1.1-4もしくは1.3.1の上位互換であれば動くはずなので、MAC,Win7,64bit,Linuxなどでの動作確認はいっさいしません。Javaは、他プラットフォームでの検証は組み合わせ爆発により不可能なので検証しません。JRE6では動く様です(セキュリティアラートが出るようですが) このアプリ、WinNT3.51とWin95でテストしていたので、ネイティブスレッド未実装のOSでも動くはずですが。

    Win7動作未確認になってますが、実際未確認です。ホントにJavaの動作環境に準じるだけです。また、大元のKissデータが入手しづらくなっているので、このソフトも公開を続ける意味があるのか分かりませんが、ソースコードは条件まもれば使えますので(GPLじゃないので注意)アニメーションスクリプトの参考にぐらいにはなるかも知れません。

  • KissLoader on Java 1.8.0 (C) 1996,1998-9,2002,2004,2018

    KOJ1.8は、Win10とJava8と10の環境で動作を確認しています。Java8/10でコンパイルが通る様に変えただけです。MACファイルフィルター無効版を入れてあります(ここ修正するならJavaFXで作り直す)。
